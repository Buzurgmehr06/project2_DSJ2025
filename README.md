# Retail Recommendation System

Проект по курсу Data Science Junior.  
Система объединяет прогноз временных рядов и рекомендательную систему для интернет-магазина.

## Цель проекта
Создать систему, которая:

- прогнозирует будущие продажи
- рекомендует товары пользователям
- решает проблему холодного старта

---

## Структура проекта

Проект состоит из двух основных модулей:

### 1. Прогноз временных рядов
Используется модель **Prophet** для прогнозирования продаж.

Функциональность:
- анализ ежедневных продаж
- выявление тренда и сезонности
- прогноз на 7, 14 или 30 дней

---

### 2. Рекомендательная система

Используется **Collaborative Filtering (user-based)**.

Этапы:
1. Строится матрица пользователь–товар
2. Считается схожесть пользователей
3. Рекомендуются товары, купленные похожими пользователями

---

### Гибридная модель

Финальная оценка рекомендации:
Hybrid Score = CF Score × Trend (за последние 30 дней)

Это позволяет:
- учитывать интересы пользователя
- учитывать актуальность товара

---

### Cold Start (новые пользователи)

Для пользователей без истории покупок:

1. Популярные товары
2. Рекомендации по категории
3. Рекомендации по интересам
4. Рекомендации по месяцу
5. Рекомендации по сезону

---

## Метрики качества

Для оценки рекомендаций используются:

- Precision@5
- Recall@5
- HitRate@5
